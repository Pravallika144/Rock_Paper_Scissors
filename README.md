# Rock, Paper, Scissors Game in C

## Description

This project implements a simple Rock, Paper, Scissors game using the C programming language. The user selects Rock, Paper, or Scissors, while the computer generates a random choice. The program compares both choices, determines the winner based on the game rules, and displays the result. It demonstrates the basic concepts of C programming, including user input, conditional statements, arrays, and random number generation.

## How It Works

The program first asks the user to choose **Rock (0)**, **Paper (1)**, or **Scissors (2)**. It then generates a random choice for the computer using the `rand()` function. Both choices are displayed, and the program compares them using conditional statements. If both choices are the same, the game ends in a tie. Otherwise, the program checks the winning combinations to determine whether the user or the computer wins and displays the appropriate message.

## Input

* User's choice:

  * **0** – Rock
  * **1** – Paper
  * **2** – Scissors

## Output

* User's choice
* Computer's choice
* Game result (User Wins, Computer Wins, or Tie)
